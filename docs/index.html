Features

Everything you need to run AI agents

🖥️

Computer Use

Give any agent hands on your desktop using native Accessibility APIs — no vision model required. Click, type, scroll, open apps, and control Chrome via CDP.

🗂️

Multi-Tab Workspace

Open unlimited agent tabs simultaneously, drag to reorder, and set per-tab working directories. Tabs and sessions are fully restored on restart.

📊

Cross-Agent Analytics

Track token usage and estimated costs across every agent and session. Daily chart, model breakdown, and agent breakdown — all stored locally.

🤖

9 AI Agent Adapters

First-class support for Claude Code, Codex CLI, GitHub Copilot CLI, Gemini CLI, OpenCode, Pi, Qwen-Code, Crush, and Hermes — each with auto-detect and install.

🧩

CoWorker Personas

Inject specialist role system prompts into any agent tab instantly. Built-in roles, online presets (Karpathy, Devin-style, Analyst), or your own custom text.

⚙️

28-Provider Settings

Configure DeepSeek, OpenAI, Anthropic, Gemini, xAI, Groq, Ollama, LM Studio, OpenRouter, and 18 more — with local inference auto-discovery.

📜

Unified History

Every conversation across all agents is captured and indexed. Search by content or agent, and restore any past session including its working directory.

⏰

Cron Scheduler

Schedule automated agent tasks with Interval, At Time, or Custom cron modes. Human-readable preview and run-count dashboard included.

🌐

Cross-Platform

Runs natively on macOS (arm64 + x64), Windows x64, and Linux x64. Full PTY terminal with xterm.js, correct resize handling, and native menus.
